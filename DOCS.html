Deployment Guide

Step-by-step instructions to deploy Kodee Preview to production on Vercel.

1

Get Anthropic API Key

Sign up at console.anthropic.com and create an API key. This powers the Claude AI preview generation.

ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxx
2

Connect GitHub to Vercel

Import the GitHub repository (soa-web/kodee-workflowaudit) into your Vercel account. Vercel will auto-detect the Next.js/React configuration.

3

Add Environment Variable

In Vercel Project Settings → Environment Variables, add your Anthropic API key. This keeps the key secure server-side.

ANTHROPIC_API_KEY=sk-ant-your-key
4

Configure Custom Domain

In Vercel Project Settings → Domains, add preview.kodee.co. Update your DNS CNAME record to point to Vercel.

preview → cname.vercel-dns.com
5

Deploy

Push to the main branch. Vercel automatically deploys on every push. Or manually trigger a deployment from the Vercel dashboard.

6

Test Full Flow

Visit https://preview.kodee.co, fill the form, and verify the preview generates correctly. Test on mobile and desktop.

Pre-Deployment Checklist

  • Add ANTHROPIC_API_KEY to Vercel environment variables
  • Connect GitHub repo to Vercel project
  • Set custom domain: preview.kodee.co → Vercel
  • Add CNAME in DNS: preview → cname.vercel-dns.com
  • Test full flow: form → loading → preview render
  • Test on mobile (form should stack to single column)
  • Test error states: empty form, API failure
  • Verify Vercel function timeout is 60s

API Reference

The /api/generate endpoint handles preview generation using Claude AI.

POST /api/generate
Request Body:
{
  "industry": "F&B / Café",
  "problem": "I want a dashboard where staff log tasks each day..."
}
Response (Success):
{
  "html": "..."
}
Response (Error):
{
  "error": "Invalid input"
}

Validation Rules

  • ✓ Industry chip must be selected (9 options available)
  • ✓ Problem textarea must be ≥ 20 characters
  • ✓ API call timeout: 60 seconds (Anthropic calls typically 20–30s)
  • ✓ Generated HTML must include <html> tag
  • ✓ All CSS in one <style> tag, all JS in one <script> tag

Local Development

Set up and run Kodee Preview on your local machine for testing and development.

# Clone the repository
git clone https://github.com/soa-web/kodee-workflowaudit.git
cd kodee-workflowaudit
# Install dependencies
pnpm install
# Create .env.local
cp .env.local.example .env.local
# Add your ANTHROPIC_API_KEY to .env.local
# Start dev server
pnpm dev
# Open http://localhost:3000
